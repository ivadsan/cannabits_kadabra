# Curso de Comunicación Asertiva

## Introducción

### Comunicación asertiva para conectar y liderar efectivamente

"Comunicar una idea"

- Claridad
- Fuerza
- Proposito

Lenguajes clave:

Verbal: Lo que dices
No verbal: Postura, gestos y mirada
Paraverbal: Tono, las pausas y silencios
